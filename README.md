# Population_Fitness

Python script for generating random atomic coordinates for small metal clusters, designed for use in global optimization and electronic structure calculations.

# Description

This code generates random atomic configurations within a defined cubic box, which can be used as initial guesses for geometry optimization of subnanoclusters (e.g., AuxFey systems).

The script was developed to support computational studies of catalytic systems, particularly for methane oxidation on bimetallic clusters.

# Features

- Random generation of atomic coordinates
- Adjustable box size
- Flexible number of atoms
- Suitable for DFT pre-optimization workflows

# Usage

Run the script:

```bash
python population_fitness.py
